# Operations

## Get Items
Get item details by ASIN list.

Inputs
- Item IDs (comma-separated)
- Resources
- Condition (optional)
- Currency of Preference (optional)
- Languages of Preference (optional)

## Search Items
Search items by keywords.

Inputs
- Keywords
- Search Index
- Item Count
- Search Items Page (optional)
- Resources
- Condition (optional)
- Currency of Preference (optional)
- Languages of Preference (optional)
 - Retry settings (optional)

## Get Browse Nodes
Fetch browse node metadata.

Inputs
- Browse Node IDs (comma-separated)
- Browse Node Resources (ancestor, children)
- Languages of Preference (optional)
